HTML Tags Definitions
<DOCTYPE!> Defines the document type
<a> Defines a hyperlink
<body> Defines the document's body
<div> Defines a section in a document
<header> Defines a header for a document or section
<img> Defines an image
<map> Defines a client-side image-map
<link> Defines a link between a document and an external source
<table> Defines a table

We used the <DOCTYPE!> tag here to define our code as an HTML document.

We used the <a> tag here to create links to other pages of our website.

We used the <body> tag here to define where the body of the Code Explanation page will start.

We used the <div> tag here to create a container for this table.

We used the h1 <header> tag here to write the heading of this page.
